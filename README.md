# Massachusetts Institute of TechnologyThis is a repo for managing machine readable artifacts for the [Massachusetts Institute of Technology APIs](http://web.mit.edu/). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [Massachusetts Institute of Technology APIs](http://web.mit.edu/).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](http://web.mit.edu/)- [Developer](http://web.mit.edu/)- [Twitter](https://twitter.com/MIT)The [APIs.json](https://github.com/api-evangelist/massachusetts-institute-of-technology/blob/master/apis.json) index provides links to all the relevant links for the Massachusetts Institute of Technology APIs, including the OpenAPI and Postman if they are present.
